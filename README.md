# 🎮 Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe** game built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates core front-end development concepts such as DOM manipulation, event handling, game logic, and responsive UI design.

---

## 📌 Features

- 🎯 Two-player gameplay (Player X vs Player O)
- 🏆 Automatic winner detection
- 🤝 Draw detection
- 🔄 Restart/New Game functionality
- 📱 Responsive design for desktop and mobile devices
- ⚡ Fast and lightweight (No external libraries)

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and responsive layout
- **JavaScript (ES6)** – Game logic and interactivity

## 🎮 How to Play

1. The game starts with **Player X**.
2. Players take turns clicking on empty cells.
3. The first player to align **three symbols** horizontally, vertically, or diagonally wins.
4. If all cells are filled without a winner, the game ends in a draw.
5. Click the **Restart** or **New Game** button to play again.

---

## 🧠 Game Logic

The game follows these steps:

- Detects user clicks on the game board.
- Places the current player's symbol (X or O).
- Checks all possible winning combinations after every move.
- Declares the winner if a winning combination is found.
- Declares a draw if all cells are filled without a winner.
- Allows the board to be reset for a new match.
